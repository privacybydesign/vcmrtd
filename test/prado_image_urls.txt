# PRADO Document Image URLs
# Pattern: https://www.consilium.europa.eu/prado/images/{DOC_ID}/{IMAGE_ID}.jpg
#
# To download all countries, you need to:
# 1. Download each country's listing page manually in your browser
# 2. Extract the image URLs from the HTML (look for links containing "/image-")
# 3. Convert detail page URLs to direct image URLs using the pattern above
#
# Example conversion:
#   Detail page: https://www.consilium.europa.eu/prado/en/AUT-AO-03001/image-367103.html
#   Direct image: https://www.consilium.europa.eu/prado/images/AUT-AO-03001/367103.jpg
